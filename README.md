# Island Stampede Website

Production-ready Next.js rebuild of Island Stampede — Tasmania's Premier Indoor Professional Bull Riding Event.

## Quick Start

```bash
npm install
npm run dev
```

`npm install` enforces npm as the package manager via a `preinstall` guard.

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Technology Stack

- **Framework:** Next.js 15 App Router
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Code Quality:** ESLint, Prettier, Husky, commitlint

## Useful Commands

- `npm run build`
- `npm run lint`
- `npm run type-check`
- `npm run milestone:status`
